# 保真度检查清单

在分析阶段列出不确定项、以及检查修复阶段核查代码时，均以此清单为准逐项确认。

---

## 结构层级

- [ ] 页面层级是否与设计稿一致

## 布局

- [ ] Auto Layout 是否正确映射为 flex / grid

## 间距

- [ ] 间距是否一致（padding、gap、margin）

## 文字

- [ ] 字号 / 行高 / 字重是否一致

## 形状

- [ ] 圆角是否一致
- [ ] 边框是否一致
- [ ] 阴影是否一致

## 资源

- [ ] 图标与文字相对位置是否一致
- [ ] 图片比例与裁切方式是否一致

## 完整性

- [ ] 是否新增了设计稿中不存在的元素
- [ ] 是否遗漏了设计稿中已有元素

## 推断项

- [ ] 所有推断项是否已单独列出（在不确定项中注明处理预案）
